## What are we all about?
Well Red Panda Studio is a growing platform for myself(Timothy) to help others to learn
the basics of languages such as Javascript, HTML, CSS, Typescript and eventually Python and
other languages that become more familiar with.

## Why?
The main reason is I want to help others I want to take what School of Code has given
me and pass it on.
Personally find it incredibly helpful to have something visual or
quick to digest.
Therefore, I want to give others with this visual learning style somewhere easy
to go to learn this way.

## How to use?
Visit https://red-panda.studio and navigate to the language you want to learn.
This will take you to a page dedicated to that language.
